Player Etiquette

온라인 포커 에티켓 가이드

다른 플레이어의 경험을 존중하는 매너는 곧 K-POKER 커뮤니티의 품질을 결정합니다.

✅ 권장 행동

  • 액션 타이밍을 일정하게 유지 (5-15초 권장)
  • 큰 핸드 승리 시 채팅으로 "GG" 또는 "GH"로 마무리
  • 상대의 좋은 콜·블러프에 "Nice play" 인정
  • 딜러(시스템)의 모든 결정에 즉시 따름
  • 네트워크 문제 발생 시 즉시 휴식 모드 신청

❌ 금지 행동

  • 의도적 슬로우 롤(이긴 패를 천천히 공개)
  • 패배한 상대 조롱 또는 욕설
  • 다계정·칩덤핑·짜고치기 (즉시 영구정지)
  • HUD 외 자동화 봇·솔버 실시간 사용
  • 다른 플레이어의 핸드 추측을 채팅으로 공개
Troubleshooting

문제 해결 가이드

자주 발생하는 기술적 문제와 해결 방법을 정리했습니다.

접속이 끊기거나 로딩이 느려요
  1. 브라우저 캐시 삭제 (Ctrl+Shift+Delete)
  2. Chrome 또는 Edge 최신 버전 사용 권장
  3. VPN/프록시 일시 비활성화
  4. 모바일이라면 Wi-Fi → LTE 전환 시도
  5. 여전히 문제 시 고객센터로 본인 IP·접속 시각 전달
테이블 입장이 안 됩니다
일반적으로 ① 잔액이 테이블 최소 바이인보다 적거나 ② 동일 IP에서 같은 테이블 입장 제한이 걸린 경우입니다. 잔액을 확인하고, 다른 테이블로 시도해보세요. 계속 발생하면 계정 상태 확인을 위해 고객센터 문의가 필요합니다.
핸드 히스토리는 어디서 확인하나요?
로그인 후 마이페이지 → 게임 기록 메뉴에서 최근 30일간의 모든 핸드를 확인할 수 있습니다. 각 핸드는 카드·베팅·결과가 시간순으로 기록되며, CSV 형식으로 다운로드해 외부 분석 도구(PokerTracker 등)에서 활용 가능합니다.
USDT 입출금 시간은 얼마나 걸리나요?
입금: TRC20 기준 평균 1-3분, ERC20 기준 평균 5-10분 (네트워크 혼잡도에 따라 변동).
출금: 신청 후 KYC 자동 검증 1-15분, 블록체인 컨펌 1-10분. 일반적으로 30분 이내 완료됩니다.
네트워크 수수료는 거래소별로 다르므로, 출금 전 거래소 수수료 정책을 확인하세요.
계정이 잠겼어요. 어떻게 해야 하나요?
비밀번호 5회 이상 오입력 시 자동으로 30분간 잠깁니다. 30분 후 자동 해제되며, 즉시 해제가 필요하면 가입 시 등록한 이메일로 본인 인증 후 고객센터에 요청하세요. 이상 활동(비정상 IP 접속 등)이 감지된 경우 보안팀이 수동 검토 후 해제까지 24시간 이내 처리됩니다.

책임 있는 플레이 (Play Responsibly)

포커는 즐거움을 위한 게임입니다. 다음 신호가 나타난다면 즉시 휴식하거나 전문가의 도움을 받으세요:

  • 예산을 초과하여 추가 자금을 빌려 플레이하고 있다
  • 잠을 줄이거나 일·학업에 지장이 있을 정도로 시간을 쏟는다
  • 가족·친구에게 플레이 사실을 숨기고 있다
  • 잃은 돈을 만회하려고 더 큰 금액을 베팅한다
  • 플레이하지 않을 때 게임 생각만 한다

도움이 필요하다면: 한국도박문제예방치유원 1336 (24시간 무료 상담), www.kcgp.or.kr
